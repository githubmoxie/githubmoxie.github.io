Node.js 中的 fs 模块

fs.readFile

  • path：文件路径（必选）

  • options：配置选项，若是字符串则指定编码格式 （可选）

    • encoding：编码格式
    • flag：打开方式

  • callback：回调函数 （必选）

    • err：错误信息
    • data：读取的数据，如果未指定编码格式则返回一个 Buffer

fs.writeFile

  • file：文件路径
  • data：写入内容
  • options：配置选项，包含 encoding, mode, flag；若是字符串则指定编码格式
  • callback：回调函数
